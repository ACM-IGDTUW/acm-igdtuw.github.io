ANALYZING COMPANY RELATED COMPLAINT TWEETS USING ABSA TO DETERMINE ITS SHORTCOMINGS

Developed with the aim of providing an analytical method to provide companies and customers with common grievances and sentiments expressed on social media using tools of sentiment analysis. Further, the project uses ABSA to specifically analyze the aspects upon which customers have specifically reviewed negatively on, upon which the company needs to improve, and the aspects the customers have reviewed positively or neutrally on, on which the company can build further specialization and a user-base. This project has made use of LSA based approaches and compared it with traditional sentiment analysis approaches like BERT.
